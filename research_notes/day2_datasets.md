# What makes a good dataset?
A strong dataset should have:
- Same person across **multiple ages**
- Same person with **different looks**
- Multiple images per person i.e. in **different pose** or in **different angle** 
- Real-world variations i.e. **different lighting conditions** 

# The FGNET Dataset (Aging dataset)
- Contains images of the same person across different ages
- Useful for studying aging effects in face recognition
- Small dataset but highly relevant for this project
